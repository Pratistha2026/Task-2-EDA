# Task 2 - Exploratory Data Analysis (EDA)

## Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Titanic dataset using Python. The analysis helps understand the dataset, identify missing values, detect outliers, analyze feature relationships, and visualize important patterns.

---

## Tools and Libraries Used

- Python
- Pandas
- Matplotlib
- Seaborn

---

## Dataset

This project uses the **Titanic Dataset**, which contains passenger information such as:

- Passenger ID
- Survival Status
- Passenger Class
- Name
- Gender
- Age
- Number of Siblings/Spouses
- Number of Parents/Children
- Ticket Number
- Fare
- Cabin
- Embarked Port

---

## Exploratory Data Analysis Performed

The following EDA steps were completed:

- Loaded the dataset using Pandas.
- Displayed the first five rows of the dataset.
- Checked the dataset dimensions.
- Displayed column names.
- Examined dataset information.
- Identified missing values.
- Generated summary statistics.
- Created Histograms.
- Created Boxplots.
- Generated a Correlation Heatmap.
- Created a Pairplot to visualize relationships among numerical features.

---

## Visualizations

The following visualizations were created:

- Histogram
- Boxplot
- Correlation Heatmap
- Pairplot

---

## Key Findings

- The dataset contains **891 rows** and **12 columns**.
- The **Age** column contains **177 missing values**.
- The **Cabin** column contains **687 missing values**.
- The **Embarked** column contains **2 missing values**.
- The **Fare** column contains several outliers.
- Most passengers belong to **Passenger Class 3**.
- Passenger Class and Fare show a noticeable negative correlation.
- Pairplots help visualize relationships between numerical features.
